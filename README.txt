		CRM- Bases de Datos 2015

Este es un sistema para el control de clientes y visualización de data social de los mismos.

Requerimientos para correr:

	- Netbeans 8.0
	- PostrgreSQL
	- MongoDB
	- Java

El proyecto se desarrollo en Java y se conecta a MongoDB y PostrgreSQL por lo que es necesario tener ejecutado
mongod.exe durante la ejecución del programa.
